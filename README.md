# NestJS Starter

This is the code base to a series of postings on my blog about my journey from Kotlin/Spring to TypeScript/Nest.js

See https://agilecoding.io/series/from-spring-to-nestjs for all details

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
